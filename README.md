# CodeAlpha Frontend Development Internship

## Overview

This repository contains the projects completed as part of the CodeAlpha Frontend Development Internship.

## Projects

### 1. Image Gallery

A responsive image gallery built using HTML, CSS, and JavaScript.

#### Features

* Responsive grid layout
* Hover effects and transitions
* Lightbox image preview
* Next/Previous navigation
* Mobile-friendly design

### 2. Calculator

A functional calculator developed using HTML, CSS, and JavaScript.

#### Features

* Addition, subtraction, multiplication, and division
* Clear screen functionality
* Keyboard support
* Responsive user interface
* Real-time calculations

### 3. Portfolio Website

A personal portfolio website showcasing skills, projects, and contact information.

#### Features

* Responsive design
* About section
* Skills section
* Projects showcase
* Contact information
* GitHub and LinkedIn links
* Smooth scrolling navigation

## Technologies Used

* HTML5
* CSS3
* JavaScript
